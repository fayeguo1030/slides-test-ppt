# 前端开发者专用的 PPT 制作页面

示例：https://fayeguo1030.github.io/slides-test-ppt/示例.html

## 使用方法

1. Fork 本项目
2. 复制 `_template.html` 到 `我的PPT.html`
3. 在 `<section>` 里面写 HTML
4. 用浏览器打开 `我的PPT.html`

## 如果你想分享给别人看

1. 开启你项目的 GitHub Page（分支选择 master）
2. 把链接放给他
3. 祈祷他没有使用 IE
